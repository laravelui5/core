    Enum LaravelUi5\Core\Enums\AbilityType

    implements BackedEnum

    Defines the four fundamental categories of Abilities within LaravelUi5. Each type expresses a distinct semantic permission scope that governs what a user may see, do, or enter across frontend and backend layers.

    Integer-backed for compact storage and stable comparison.

    Use → Access to frontend views, dialogs, or routes Act → Execution of backend operations or actions See → Visibility control for UI elements Access → Permission to enter system-level artifacts (apps, dashboards, reports, cards, tiles, KPIs, dialogs, resources)
