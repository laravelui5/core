# LaravelUi5 Core

Foundation layer for integrating [OpenUI5](https://openui5.org) into [Laravel](https://laravel.com). Provides runtime, routing, OData integration, and developer tooling for building enterprise UI5 apps in a Laravel-native workflow.

## Features

- **Artifact system** — Applications, Libraries, Cards, Dashboards, Actions, Resources, and more as first-class Laravel components
- **OData v4** — Built on [laravelui5/odata](https://github.com/laravelui5/odata) (MIT), every UI5 app is an OData service out of the box
- **Scaffold generators** — Artisan commands to generate apps, libraries, cards, tiles, actions, resources, and dashboards
- **Contract-first design** — Clean interfaces and abstract bases for every artifact type

## Requirements

- PHP ^8.4
- Laravel 12+

## Installation

LaravelUi5 Core is distributed via private Composer registry. Register at [laravelui5.com](https://laravelui5.com) to get access.

```bash
composer require laravelui5/core
```
